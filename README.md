# Linux Toolkit

A personal Linux knowledge base built while learning Linux administration, infrastructure, and system operations.

The goal of this repository is to document practical Linux commands, troubleshooting techniques, Bash scripts, and operational knowledge gained through hands-on practice.

## Repository structure

```
commands/
scripts/
notes/
```

## Commands

- Filesystem
- Users & Permissions
- Processes
- Networking
- Logs
- Systemd
- SSH

## Scripts

- system_check.sh
- disk_usage.sh
- service_status.sh

## Goals

- Build a structured Linux knowledge base
- Practice Bash scripting
- Improve troubleshooting skills
- Prepare for Infrastructure / IT Support / DevOps roles

# Filesystem Commands

This document contains practical Linux filesystem commands that I use while learning Linux administration and infrastructure.

---

## pwd

**Purpose**

Print the current working directory.

**Example**

```bash
pwd
```

---

## ls

**Purpose**

List files and directories.

**Examples**

```bash
ls
ls -l
ls -la
```

---

## cd

**Purpose**

Change the current directory.

**Examples**

```bash
cd /home
cd ..
cd ~
```

---

## mkdir

**Purpose**

Create a new directory.

**Example**

```bash
mkdir test_folder
mkdir -p project/scripts
```

---

## touch

**Purpose**

Create an empty file.

**Example**

```bash
touch notes.txt
```
